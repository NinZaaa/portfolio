# Bidhan Dhakal Personal Brand Website

Long-term personal brand website built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, Lucide, next-themes, and shadcn/ui.

## Project Status
- Foundation setup complete.
- Website UI/content implementation intentionally not started yet.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Framer Motion
- Lucide React
- next-themes

## Scripts
- `npm run dev`: start local dev server
- `npm run build`: create production build
- `npm run start`: run production server
- `npm run lint`: lint code with ESLint
- `npm run typecheck`: run TypeScript checks
- `npm run check`: run lint + typecheck

## Environment
1. Copy `.env.example` to `.env.local`.
2. Update values as needed.
